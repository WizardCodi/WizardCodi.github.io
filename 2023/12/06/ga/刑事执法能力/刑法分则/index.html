公专要点-刑事执法能力-刑法分则

2023-12-06
笔记

亲告罪

特点

自诉案件，不告就不管。归法院管，公安不管。

罪名

  • 诽谤罪（严重危害社会秩序和国家利益的除外）
  • 侵占罪（绝对亲告罪）
  • 侮辱罪（绝对亲告罪）
  • 暴力干涉婚姻自由罪（致人死亡除外）
  • 虐待罪（致人重伤、死亡的除外）

盗窃罪

定义

将他人占有的财物转为自己占有。不要求具有秘密性，可公开进行。

行为结构

将他人合法或非法占有的财物通过和平的手段转为自己占有。

行为方式

  • 普通盗窃：
    要求数额较大。一千～三千元以上为数额较大（根据当地经济情况裁定）
  • 多次盗窃：
    二年内三次以上，不限数额。
  • 入户盗窃：
    “户”指相对隔离的生活场所。不限数额。
  • 携凶器盗窃：
    要有对人使用凶器的意图。凶器只要携带就算，不限数额。
  • 扒窃：
    在公共场所、交通工具上盗窃他人随身携带的财物。不限于体积微小的财物。

既遂问题

  • 财物很大，犯罪人转移出场所才是既遂。
  • 财物很小，犯罪人藏于身上或隐蔽处就是既遂。

抢夺罪

定义

以非法占有为目的，当场直接夺取他人紧密占有财物的行为。

行为结构

对物实施暴力，对人有危险行为。

与抢劫罪的区别在于：抢劫罪对人也实施暴力。

**携凶器抢夺按抢劫罪规定处罚。**凶器可以由第三人携带。

飞车抢夺问题

  • 夺取他人财物时因被害人不放手而强行夺取的：抢劫罪；
  • 驾车逼挤、撞击或强行逼倒他人夺取财物：抢劫罪；
  • 明知会致人死亡仍强行夺取并放人财物持有人产生轻伤以上后果：抢劫罪。

抢劫罪

行为结构

实施暴力、胁迫等手段，足以压制被害人反抗 -> 被害人因无法反抗放弃财物 -> 行为人夺取财物。

行为与目的同时存在原则：暴力行为的目的必须是非法占有。

行为对象

  • 财物占有者
  • 财物占有的辅助者
  • 财物保护者
  • 不包括无关的第三人

数罪

  • 为了当场取得财物，当场使用暴力故意将人杀死，成立抢劫罪致人死亡。
  • 因其他原因故意杀人后产生非法占有财物意图，认定为故意杀人罪 + 盗窃罪（或侵占罪）
  • 抢劫后，为了灭口而杀人的，成立抢劫罪 + 故意杀人罪

转化抢劫

  • 客观条件：
    盗窃、诈骗、抢夺时为窝藏赃物、抗拒抓捕或毁灭罪证，当场使用暴力或以暴力威胁。（要求暴力程度较高）
  • 前提条件：
    犯盗窃、诈骗、抢夺罪。
  • 年龄规定：
    已满14未满16者不构成转化抢劫。

抢劫加重情形

  • 入户抢劫：
    **集体宿舍、宾馆不算户。**要有住宅安宁权，并且要求是非法进入。
  • 公共交通工具上的抢劫：
    • 包括对运行途中的机动交通工具加以拦截后，对上面的人员实施抢劫；
    • 在未运营中的大中型交通工具中针对司售、乘务人员抢劫的不成立；
    • 小型出租车不成立；
  • 抢劫银行的
  • 多次抢劫或数额较大（三次以上）
  • 致人重伤或死亡
  • 冒充军警
  • 持枪抢劫
    • 使用枪支
    • 向被害人展示持有佩戴的枪支并实施抢劫
    • 要求真枪，不要求实弹
  • 抢劫军用、抢险救灾物资

绑架罪

行为结构

以向第三人提出不法要求为目的，使用暴力、胁迫或麻醉方法劫持或以实力控制他人 -> 向第三人提出不法要求
绑架的对象要足以使得第三人产生担忧。

既遂

带着胁迫第三人目的，控制了人质就是绑架罪的既遂。
若人质不引起第三方担忧，则未遂。

非法债务问题

非法债务指“赌债”、“高利贷”等等。

  • “不还债，不放人” 仅构成非法拘禁；
  • “不还债，伤人质” 构成绑架罪。
    如果以索取合法债务为目的绑架他人，构成非法拘禁。

诈骗罪

行为结构

进行欺诈行为 -> 被害人产生错误认识 -> 被害人基于错误认识处分财产 -> 行为人取得财产 -> 被害人受到财产损失
欺骗程度要求能使得一般人产生错误认识。

信用卡诈骗犯罪

  • 客体
    信用卡管理制度和公私财产所有权
  • 行为手段
    • 违反信用卡管理法规
    • 使用伪造信用卡，或以虚假身份骗领的信用卡
    • 使用作废的信用卡
    • 冒充他人信用卡
    • 恶意透支
  • 盗窃信用卡并使用：以盗窃罪定罚。
    明知是假卡/做废卡，仍盗窃并对人使用，认定为信用卡诈骗罪。

危险驾驶罪

情形

  1. 机动车追逐竞驶且情节恶劣；
    • 不要求2人以上，一人也可以追逐竞驶。
    • 要求情节恶劣。
  2. 道路上醉驾；
    • 不要求情节恶劣；
    • 没有公共危险行为的不可能成立本罪。
  3. 从事校车或旅客运输，严重超载、超速的；
  4. 违反危险化学品安全管理规定运输危险化学品，危及公共安全的。

交通肇事罪

所有的交通肇事罪都是过失犯罪。

行为结构

在交通运输过程中违反交通法规，导致发生重大事故。

成立条件

主体条件

  • 包括从事交通运输的人员（乘客、行人等）；
  • 交通工具不限于机动车辆；
  • 单位主管人员、机动车辆所有人/承包人，指使、强令他人违章驾驶造成重大事故的，以交通肇事罪论处；
  • 单位主管人员、机动车辆所有人/承包人，指使、强令肇事者逃逸，使得被害人因得不到救助而死亡的，以交通肇事罪的共犯论处。

时空条件

  • 公共交通管理范围（公路/水上/铁路/航空）
  • 不包括封闭小区

情形

  • 酒驾毒驾机动车辆
  • 无证驾驶机动车辆
  • 明知机动车辆安全装置不全或安全装置失灵的
  • 明知无牌证或已报废的机动车辆
  • 严重超载
  • 逃逸的

加重处罚情形

  • 事后逃逸（构成肇事罪的基础上升格处理，3～7年）：
    • 主观动机：逃避法律追究
    • 举例：
      • A肇事后，站在原地不救助被害人，属逃逸
      • 救助被害人后逃跑，不属逃逸
      • 直接撞死了被害人后逃跑，由于无救治必要，不属逃逸
  • 因逃逸致人死亡（7～15年）
  • 将被害人带离现场后隐藏或遗弃：
    肇事后为逃避法律追究，将被害人带离事故现场后隐藏或遗弃，使其因无救助而死亡或严重伤残，分别以故意杀人罪和故意伤害罪论处。
    若之前的行为已经构成交通肇事罪，则数罪并罚
  • 事后抛尸：
    • 事后以为被害人已死，为隐匿罪证而抛尸致死，以交通肇事罪和过失致人死亡罪数罪并罚

高空抛物罪

从建筑物或其它高空抛物，情节严重的，处一年以下有期、拘役或管制，并处或单处罚金。
有前款行为，同时构成其它犯罪的，依处罚较重的规定定罪处罚。
为伤害、杀害特定人员实施上述行为的，依故意伤害罪/杀人罪定罪处罚。
